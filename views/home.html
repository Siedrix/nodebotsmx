Robots powered by JavaScript

Photo: Chris Waring

Así es... si tienes algo de experiencia en JavaScript, puedes usar tus poderes para controlar máquinas. Desde hacer parpadear un led hasta manejar un carrito, jugar con tu gato de manera remota :-], y torear a un quadcopter...

Alrededor del mundo, existen eventos donde las personas se reunen para hackear Arduinos, Raspberry Pis, NodeCopters, robots y otros artefactos.

Si no tienes idea de como usar un LED, no te preocupes, ese día habrán personas que puedan ayudarte a entender que es y como puede ser programado.

Pretendemos tener un taller, en la cual construiremos un pequeño carrito usando Node, Arduino y Johnny-Five.

Iniciando con Node y Johnny-Five

Instalar Node.js y descarga el módulo de johnny-five. Necesitarás una placa Arduino con el sketch de Firmata cargado, lo cual es muy fácil de hacer. Después de tener tu Arduino con Firmata correctamente cargado, ya puedes iniciar a hackear tu Arduino como campeón!. NodeBots hace que el proceso de programación del hardware sea muy similiar a la programación de una página web moderna. En serio, es muy fácil!.

Incluso, si no tienes un Arduino a la mano, puedes iniciarte utilizando nodebot-workshop el cual, es un módulo de node que te enseña lo básico de Johnny-Five, a través de una serie de retos:

¿De dónde viene NodeBots?

El concepto de NodeBots surgió inicialmente con el creciente interés por la creación y manipulación de hardware dentro de la comunidad emprendedora de node js.

Chris Williams es el creador del módulo conocido como node-serialport, el cual permite a los programadores de JS controlar y dirigir el mundo físico. Desde entonces, el hardware ha sido el punto principal de cada conferencia de JS (JSConf), maravillando audiencias con cualquier cosa, desde maquinas de humo (usadas comunmente en los antros) hasta robots que caminan. Hoy es el momento para que cualquiera pueda iniciarse. Now is time for the rise of JS Robotics.
